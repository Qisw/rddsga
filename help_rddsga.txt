****** Help of the commands:


Syntax
		
		syntax varlist [if] [in] ,  weight(varlist) treat(varlist) bw(string) [c(real 0) 
		x(varlist) covs(varlist) fe(varlist) cluster(string) FirstStage ReducedForm  
		ivreg all spline(string) addnamtex(string) latex dir(string)
		bdec(int 3) labelt0(string)]


Options 	Description

weight		Weight of each observation obtained from the p-score. The number of weights 
		specified indicates how many columns will be displayed in the table. This option 
		is required. 

treat	        treatment or subgroup variable. The option is required.

bw  		specifies the bandwidth in a matrix. The number of columns in the 
		matrix has to be equal to the number of weights. This option is required.
			
c	        cut-off point. The default is 0. 

x		instrumentalized variable. It is neccessary for the First Stage.

covs		covariates included as controls in the regressions.

fe 		categorical variable to be absorbed (fixed effect).

cluster		categorical variable to cluster standard errors.

FirstStage	compute first stage regression.

ReducedForm	compute reduced form regression.

ivreg		compute instrumental variables regression. 

all		compute first stage, reduced form and instrumental variables regression.
		"all" has priority according to "ivreg"/"ReducedForm"/"FirstStage"
	
spline		specifies the spline in a string. The number of strings has to be 
		equal to the number of weights. The options are L (linear) or Q 
		(quadratic). Example: spline(L Q L). Defaults is (L L L)

addnamtex	adds a name to the latex output. Default is G0 and G1...

latex 		exports a latex table with the balance improvement.
		
dir 		specifies the directory where each latex output is placed.
	
bdec		specifies the number of decimal places reported for all estimates. 
		The default value for bdec is 3.

labelt0         specifies the label of subgroup "0" or control for formatting table. 		
		

Example:

matrix input bw=(4,10,5.43)
rddsga sh_licitacion dis_cutoff2, latex addnamtex(AU_weight) spline(L Q L) treat(high_direct) all \\
x(I_CURaudit) fe(gpaoXuceXrk) cluster(gpaoXuceXrk) covs(p1 p2 p3 sh_licitacionPRE2 sh_directoPRE1 \\
sh_licitacionPRE1 sh_directoPRE2 size_PRE1 size_PRE2 I_PREaudit high_direct) weight(weight4 weight10 \\
bw(bw) labelt0(1\{DN share $<$ p(50)\})

This line generates the same result of the paper.
